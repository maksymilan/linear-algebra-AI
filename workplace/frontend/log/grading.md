# 题目
给定一个变换矩阵 A 和一个向量 x：

A = [[2, -1], [1, 3]]
x = [[3], [2]]

1. 计算向量 x 经过矩阵 A 变换后的新向量 x' (即 x' = Ax)。
2. 描述这个变换在几何上大致是如何影响基向量 î = [1, 0] 和 ĵ = [0, 1] 的。

请计算以下3x3矩阵 B 的逆矩阵 B⁻¹：

B = [[1, 2, 3], [0, 1, 4], [5, 6, 0]]

# 解答
1. 计算 x' = Ax:
x' = [[2, -1], [1, 3]] * [[3], [2]]
   = [[(2*3) + (-1*2)], [(1*3) + (3*2)]]
   = [[6 - 2], [3 + 5]]
   = [[4], [8]]

2. 几何影响：
变换后的 î' 是矩阵A的第一列 [2, 1]。
变换后的 ĵ' 是矩阵A的第二列 [-1, 3]。
这个变换将基向量 î 拉长并旋转，同时也将基向量 ĵ 拉长并旋转到了第二个象限。

要求矩阵B的逆矩阵，首先需要计算其行列式 det(B)。
det(B) = 1 * (1*0 - 4*6) - 2 * (0*0 - 4*5) + 3 * (0*6 - 1*5)
       = 1 * (-24) - 2 * (-20) + 3 * (5)
       = -24 + 40 + 15
       = 31

接下来计算伴随矩阵 adj(B)。
余子式矩阵 C = 
[[-24, 20, -5],
 [-18, -15, 4],
 [5, 4, 1]]

转置余子式矩阵得到伴随矩阵 adj(B) = Cᵀ =
[[-24, -18, 5],
 [20, -15, 4],
 [-5, 4, 1]]

最后，B⁻¹ = (1/det(B)) * adj(B)
B⁻¹ = (1/31) * [[-24, -18, 5], [20, -15, 4], [-5, 4, 1]]
